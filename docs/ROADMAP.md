# Buff Helper Roadmap

## Vision

Buff Helper est un assistant universel de configuration GNU/Linux.

Il aide les débutants à construire leur système idéal après une installation Linux ou directement depuis Calamares.

Une distribution (BuffOS, Kubuntu, Fedora KDE, etc.) n'a qu'à fournir ses profils.
